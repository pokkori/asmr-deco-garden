# 効果音ファイル一覧

下記ファイルをこのフォルダに配置してください。
無料素材サイト例: https://soundeffect-lab.info / https://www.zapsplat.com

| ファイル名             | 用途                              | 推奨時間  |
|----------------------|----------------------------------|---------|
| collect_common.mp3   | Common アイテム収集時              | ~0.3秒  |
| collect_rare.mp3     | Rare アイテム収集時（シャリーン）   | ~0.5秒  |
| collect_epic.mp3     | Epic アイテム収集時（魔法音）      | ~0.6秒  |
| collect_legendary.mp3| Legendary 収集時（ファンファーレ）  | ~1.0秒  |
| reveal.mp3           | スクラッチ80%到達・アイテム出現時  | ~0.8秒  |
| place.mp3            | ガーデンにアイテム配置時           | ~0.2秒  |

## 開発中のダミー対応

音源が無くてもアプリは動作します（CollectionScreen の playSound が例外を握りつぶします）。
開発中は音無しで動作確認できます。
